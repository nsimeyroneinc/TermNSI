    La notation d'un QCM de NSI est la suivante : 3 points par rÃ©ponse correcte, -1 point par rÃ©ponse incorrecte et 0 si l'on n'a pas rÃ©pondu

    ComplÃ©ter la fonction correction_QCM_Alice(reponses_Alice,reponses_valides) qui, Ã  partir des dictionnaires reponses_Alice et reponses_valides passÃ©es en paramÃ¨tres renvoie le nombre de points obtenus au QCM par Alice.

    ğŸ Script Python
    def correction_QCM_Alice(reponses_Alice,reponses_valides):
    note=0
    for question in reponses_valides.keys():
        if question in reponses_Alice:
            if reponses_Alice[question]==reponses_valides[question]:
                note+=3
            elif reponses_Alice[question] !=reponses_valides[question]:
                note-=1
    return note
